Single-line text input. Pair with `Field` for a label, hint and error.

```jsx
<Input placeholder="رقم الاشتراك" />
<Input invalid value="abc" />
<Input readOnly value="CS0001" />
```

`invalid` switches to the crimson error treatment. Focus paints the crimson ring and a faint blue wash; native `readOnly`/`disabled` grey out.
