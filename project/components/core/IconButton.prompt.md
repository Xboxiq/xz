Square, icon-only button for topbar actions and toolbars — navy tint that warms to crimson on hover.

```jsx
<IconButton icon="notifications" label="الإشعارات" />
<IconButton icon="search" size="lg" />
<IconButton icon="more_vert" bare label="المزيد" />
```

Sizes `sm`/`md`/`lg`. `bare` drops the tinted background. Always pass `label` for accessibility.
