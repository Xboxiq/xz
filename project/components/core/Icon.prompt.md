Material Symbols Outlined glyph wrapper — the hub's single icon primitive. Pass a ligature name.

```jsx
<Icon name="bolt" />
<Icon name="receipt_long" size="lg" filled />
<Icon name="warning" bold />
```

Sizes `sm`/`md`/`lg`/`xl`. `filled` flips the FILL axis; `bold` sets weight 700. Material Symbols loads from Google Fonts via `styles.css`, so any [Material Symbols](https://fonts.google.com/icons) name works.
