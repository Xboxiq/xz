Small uppercase-mono status pill — system states like ACTIVE, PENDING, OVERDUE.

```jsx
<Tag tone="success" dot>نشط</Tag>
<Tag tone="warn">قيد المراجعة</Tag>
<Tag tone="crimson" dot>عاجل</Tag>
```

Tones: `default` (navy), `success`, `warn`, `err`, `info`, `crimson`, `gold`. `dot` adds the haloed status dot. Reads as a label-caps badge thanks to the mono font + letter-spacing.
