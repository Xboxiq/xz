The department identity square. The four sections each own a fixed color — always use this so a badge reads as its department.

```jsx
<SectionBadge section="CS" />   {/* الاشتراكات — blue   */}
<SectionBadge section="CB" />   {/* الفواتير — teal    */}
<SectionBadge section="CT" />   {/* الفنية — amber     */}
<SectionBadge section="CA" />   {/* الشكاوى — red      */}
```

`CS` blue, `CB` teal, `CT` amber, `CA` red. Defaults its label to the code; pass children to override.
