Filter / selection chip — a row of these heads up most list views.

```jsx
<Chip active>الكل</Chip>
<Chip icon="apartment">منزلي</Chip>
<Chip section="CT" active>الفنية</Chip>
```

`active` fills the chip (navy by default). Pass `section` (`CS`/`CB`/`CT`/`CA`) to tint the active fill with that department's color. `icon` takes a Material Symbols name.
